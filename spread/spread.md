
спред идентичен синтаксически рест оператору, но выполняет противоположное действие 

`
const sum = (...numbers) => {
  let result = 0;
  for (let num of numbers) {
    result += num;
  }
  return result;
};

const numbers = [1, 7, 4];
sum(...numbers); // 12

`
